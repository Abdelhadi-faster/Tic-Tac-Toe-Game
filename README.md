# ❌⭕ لعبة XO (Tic-Tac-Toe) باستخدام Windows Forms

## 🧾 الوصف
هذا مشروع لعبة XO (إكس-أو) تم تطويره باستخدام لغة C# و Windows Forms.  
اللعبة تعمل بين لاعبين (Player 1 و Player 2)، حيث يقوم كل لاعب بوضع رمزه (X أو O) على لوحة اللعب، والفائز هو من يتمكن من تكوين 3 رموز متتالية.

---

## 🎮 طريقة اللعب
- اللاعب الأول يبدأ باستخدام الرمز (X)
- اللاعب الثاني يستخدم الرمز (O)
- يتم الضغط على أحد المربعات لوضع الرمز
- لا يمكن اختيار نفس المربع مرتين
- عند اكتمال صف أو عمود أو قطر يتم إعلان الفائز
- إذا امتلأت كل الخانات بدون فائز → تعادل

---

## 🚀 المميزات
- واجهة رسومية باستخدام Windows Forms
- عرض دور اللاعب الحالي
- تحديد الفائز تلقائيًا
- تمييز الصف الفائز بلون مختلف
- عرض رسالة عند انتهاء اللعبة
- زر لإعادة تشغيل اللعبة

---
